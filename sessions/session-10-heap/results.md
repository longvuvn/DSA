# Buổi 10: Kết quả

## Thực hành

### Bài 1: Cài đặt heap

- Input: buildHeap `[3,1,6,5,2,4]`
- Output: max-heap top = `6`.

### Bài 2: Top-K

- Input: `[3,2,1,5,6,4]`, k=2
- Output: `[6,5]`.

## Bài tập

1. **Merge k sorted lists**
   - Input: `[[1,4,5],[1,3,4],[2,6]]`
   - Output: `[1,1,2,3,4,4,5,6]`.

2. **Median of stream**
   - Input: `2,1,5,7,2,0,5`
   - Output: `2,1.5,2,3.5,2,2,2`.

3. **Priority queue update key**
   - Ý tưởng: decrease-key + heapify.

4. **Heap sort**
   - Input: `[4,10,3,5,1]` → Output: `[1,3,4,5,10]`.
