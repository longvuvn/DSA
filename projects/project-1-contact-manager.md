# Project 1: Contact Manager (Danh bạ)

## Mục tiêu

Xây dựng một ứng dụng quản lý danh bạ đơn giản, hỗ trợ thêm/xóa/sửa/tìm kiếm.

## Kiến thức áp dụng

- Linked List (danh sách liên kết)
- Hash Table (ánh xạ theo tên/số điện thoại)
- Phân tích độ phức tạp thao tác

## Yêu cầu chức năng

1. Thêm liên hệ (name, phone, email).
2. Xóa liên hệ theo tên hoặc số.
3. Cập nhật thông tin liên hệ.
4. Tìm kiếm theo tên (exact/prefix).
5. In toàn bộ danh bạ theo thứ tự thêm vào.

## Gợi ý thiết kế

- Dùng Linked List để lưu theo thứ tự thêm.
- Dùng Hash Table để tra cứu nhanh theo tên.

## Tiêu chí hoàn thành

- Mỗi thao tác có độ phức tạp rõ ràng.
- Có kiểm thử với dữ liệu mẫu tối thiểu 20 liên hệ.
