# DSA Roadmap (Cấu trúc dữ liệu & Giải thuật)

Chào bạn! Repo này chứa lộ trình học DSA từ cơ bản đến nâng cao, kèm theo tài liệu cho từng buổi học: **lý thuyết**, **thực hành**, và **bài tập**. Mục tiêu là học chắc nền tảng và có dự án thực tế để luyện kỹ năng.

## Cấu trúc thư mục

- `roadmap.md`: Lộ trình học tổng quan theo buổi.
- `sessions/`: Tài liệu theo từng buổi học.
  - `ly-thuyet.md`: Giải thích khái niệm rõ ràng, dễ hiểu (có minh họa).
  - `thuc-hanh.md`: Hướng dẫn thực hành từng bước.
  - `bai-tap.md`: Bài tập luyện tập từ cơ bản đến nâng cao.
- `projects/`: Mô tả các project thực hành.

## Gợi ý cách học

1. Đọc `ly-thuyet.md` trước.
2. Làm theo `thuc-hanh.md` (tự code theo ngôn ngữ bạn chọn).
3. Làm `bai-tap.md` và tự đánh giá.
4. Sau vài buổi, bắt đầu làm project để tích hợp kiến thức.

## Lưu ý

Bạn có thể dùng **C++**, **Python**, hoặc **Java** để luyện tập. Các bài thực hành được viết theo hướng thuật toán nên dễ chuyển đổi ngôn ngữ.
