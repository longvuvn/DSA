# Java Practice (DSA)

Thư mục này cung cấp **template Java** cho từng buổi học để bạn code ngay trong VS Code, không cần tạo project hay IDE khác.

## Cách chạy nhanh (mỗi buổi là một thư mục riêng)

1. Mở terminal tại thư mục buổi học, ví dụ `session-01`.
2. Biên dịch và chạy:

```pwsh
javac Main.java
java Main
```

## Lưu ý

- Mỗi buổi là một chương trình độc lập.
- File luôn là `Main.java` để bạn chạy nhanh.
- Nếu muốn chia nhỏ file, bạn có thể tạo thêm class không public trong cùng thư mục.
