# Buổi 09: Kết quả

## Thực hành

### Bài 1: Cài đặt BST

- Input: insert `[8,3,10,1,6]`
- Output: search(6) → `true`, delete(3) → cây hợp lệ.

### Bài 2: Kiểm tra BST hợp lệ

- Ý tưởng: inorder phải tăng dần hoặc dùng min/max.
- Output mẫu: cây mẫu → `true`.

## Bài tập

1. **k nhỏ nhất trong BST**
   - Input: k=3 → Output: `6`.

2. **LCA trong BST**
   - Input: node 1 và 6 → Output: `3`.

3. **BST → sorted list**
   - Output mẫu: `[1,3,6,8,10]`.

4. **AVL rotation**
   - Ý tưởng: LL → right rotation, RR → left rotation.
