# Buổi 05: Kết quả

## Thực hành

### Bài 1: Giai thừa & Fibonacci

- Ý tưởng: đệ quy và vòng lặp để so sánh.
- Input: n=5 → fact=120.
- Input: n=6 → fib=8.

### Bài 2: Sinh hoán vị

- Ý tưởng: backtracking, swap tại index.
- Input: `[1,2,3]`
- Output: 6 hoán vị (ví dụ: `[1,2,3]`, `[1,3,2]`, ...).

## Bài tập

1. **Tính tổng 1..n**
   - Input: n=4 → Output: `10`.

2. **In dãy nhị phân độ dài n**
   - Input: n=3 → Output: `000..111` (8 chuỗi).

3. **N-Queens**
   - Input: n=4 → Output: `2` nghiệm.

4. **Subset sum**
   - Input: `[2,3,7,8,10]`, K=11 → Output: `true`.
