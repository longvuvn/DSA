# Buổi 07: Kết quả

## Thực hành

### Bài 1: Binary Search

- Ý tưởng: dùng 2 con trỏ l/r và mid.
- Input: `[1,3,5,7,9]`, target=7 → Output: index `3`.

### Bài 2: Hash Map đếm tần suất

- Input: `"abcaab"`
- Output: `{a=3, b=2, c=1}`.

## Bài tập

1. **Lower bound**
   - Input: `[1,2,4,4,5]`, x=4 → Output: index `2`.

2. **Two Sum**
   - Input: `[2,7,11,15]`, target=9 → Output: `(0,1)`.

3. **Subarray sum equals k**
   - Input: `[1,2,3]`, k=3 → Output: `2`.

4. **Longest consecutive sequence**
   - Input: `[100,4,200,1,3,2]` → Output: `4`.
